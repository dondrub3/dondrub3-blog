# 3P 博客文章模板使用指南

## 快速开始

### 创建新文章

```bash
# 使用 Hugo 命令创建新文章（自动使用模板）
hugo new content posts/YYYY-MM-DD-文章标题.md

# 示例
hugo new content posts/2026-03-10-example-3p-analysis.md
```

### 手动创建

复制以下模板内容到新的 `.md` 文件：

```markdown
---
title: "文章标题"
date: 2026-03-10T16:00:00+08:00
draft: false
tags: ["3P分析", "标签1", "标签2", "标签3"]
categories: ["交易3P分析"]
summary: "文章摘要，简要描述文章内容，用于列表页显示"
---

> 文章引言/导语，用 quote 格式突出核心观点。这句话会显示在文章列表中。

---

## 导语

文章背景介绍，说明热点事件或话题...

### 1.1 子标题

分析内容...

**关键要点：**
- 要点1

- 要点3

### 1.2 子标题

更多分析...

---

## 二、Psychology：心理机制与行为分析

### 2.1 子标题

心理分析内容...

### 2.2 子标题

更多心理分析...

### 3.1 子标题

### 3.2 子标题

*本文写于2026年3月10日，基于公开信息分析，不构成投资建议。*
*3P框架：Physics（物理/规律）、Psychology（心理/行为）、Philosophy（哲学/价值观）*
```

## Front Matter 字段说明

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| `title` | ✅ | 文章标题 | `"当油价冲破100美元：..."` |
| `date` | ✅ | 发布日期 | `2026-03-09T14:00:00+08:00` |
| `draft` | ✅ | 是否为草稿 | `false` (发布) / `true` (草稿) |
| `tags` | ✅ | 文章标签 | `["油价", "能源危机", "3P分析"]` |
| `categories` | ✅ | 文章分类 | `["交易3P分析"]` 或 `["决策3P分析"]` |
| `summary` | 可选 | 文章摘要 | 用于列表页显示 |
| `author` | 可选 | 作者 | `"东周"` |

### 分类选择

- `交易3P分析` - 与交易、投资相关的分析
- `决策3P分析` - 与决策、战略相关的分析
- `哲学` - 纯哲学思考
- `心理学` - 纯心理学分析

### 常用标签

**主题类：**
- `3P分析` - 必加
- `油价`、`黄金`、`AI`、`地缘政治`
- `交易心理`、`风险管理`

**类型类：**
- `投资分析`、`技术分析`、`基本面分析`
- `热点解读`、`深度分析`

## 文章结构规范

### 1. Front Matter（元数据）

```yaml
---
title: "文章标题"
date: 2026-03-10T16:00:00+08:00
draft: false
tags: ["3P分析", "..."]
categories: ["交易3P分析"]
summary: "..."
---
```

**⚠️ 注意：**
- `---` 必须独占一行
- 字段缩进使用空格（不要用 Tab）
- 字符串值用双引号包裹
- 列表用方括号 `[]`

### 2. 引言（Quote）

```markdown
> 文章核心观点，一句话总结
```

- 必须紧跟 front matter 后面
- 用 `>` 引用格式
- 简洁有力，突出核心洞察

### 3. 导语

```markdown
## 导语

背景介绍...
```

- 说明热点事件或话题背景
- 引出为什么要用3P框架分析

### 4. 三个维度分析

```markdown
## 一、Physics：物理规律与系统分析

### 1.1 子标题
内容...

## 二、Psychology：心理机制与行为分析

### 2.1 子标题
内容...

## 三、Philosophy：哲学思考与价值反思

### 3.1 子标题
内容...
```

- 每个维度必须有
- 使用二级标题 `##`
- 子标题使用三级标题 `###`

### 5. 结语（必须有行动建议）

```markdown
## 结语：给读者的行动建议

### 核心洞察回顾

- **Physics**：物理层面的关键发现...
- **Psychology**：心理层面的核心洞察...
- **Philosophy**：哲学层面的深度思考...

### 具体行动建议

**如果你是投资者：**
- 建议1：...
- 建议2：...
- 建议3：...

**如果你是决策者：**
- 建议1：...
- 建议2：...
- 建议3：...

**如果你是普通读者：**
- 建议1：...
- 建议2：...

### 最后的提醒

> 记住东周的3P原则：科学 ≠ 真理，模型有边界，保持谦卑与好奇。

在不确定的世界中，保持清醒的头脑，做好应对各种可能的准备。
```

**结语要求：**
- ✅ 回顾三个维度的核心洞察
- ✅ 给不同读者群体具体建议（投资者/决策者/普通读者）
- ✅ 用 quote 强调核心原则
- ✅ 结尾有鼓舞或提醒

### 6. 文末声明

```markdown
*本文写于2026年3月10日，基于公开信息分析，不构成投资建议。*
*3P框架：Physics（物理/规律）、Psychology（心理/行为）、Philosophy（哲学/价值观）*
```

## 常见错误

### ❌ 错误1：正文重复标题

```markdown
---
title: "文章标题"
---

# 文章标题  ← 错误！front matter 已经有标题了

> 引言
```

### ✅ 正确

```markdown
---
title: "文章标题"
---

> 引言（直接从 quote 开始）
```

### ❌ 错误2：缺少 front matter

```markdown
# 文章标题

正文...
```

### ✅ 正确

```markdown
---
title: "文章标题"
date: 2026-03-10
---

> 引言
```

### ❌ 错误3：front matter 格式错误

```markdown
---
title: 文章标题    ← 错误！字符串必须用引号
date: 2026-03-10   ← 错误！需要完整格式
tags: 标签1        ← 错误！必须是列表格式
---
```

### ✅ 正确

```markdown
---
title: "文章标题"
date: 2026-03-10T16:00:00+08:00
tags: ["标签1", "标签2"]
---
```

## 发布流程

1. **创建文章**
   ```bash
   hugo new content posts/2026-03-10-article-title.md
   ```

2. **编辑内容**
   - 填写 front matter
   - 写入正文（不要重复标题）

3. **本地预览**
   ```bash
   hugo server -D
   ```

4. **构建站点**
   ```bash
   hugo --minify
   ```

5. **提交部署**
   ```bash
   git add -A
   git commit -m "Add new article: 文章标题"
   git push origin main
   ```

## 结语示例（参考）

### 好的结语示例

```markdown
## 结语：给读者的行动建议

### 核心洞察回顾

- **Physics**：油价破百不是投机泡沫，而是物理供给约束的硬着陆。能源转型需要时间，而地缘政治风险不等人。
- **Psychology**：恐惧与贪婪的钟摆正在摆动，通胀预期可能自我实现。保持理性比追逐热点更重要。
- **Philosophy**：现代性建立在廉价能源之上，当能源不再廉价，我们需要重新定义"进步"和"好生活"。

### 具体行动建议

**如果你是投资者：**
- 不要盲目追高能源板块，警惕短期情绪过热
- 关注能源转型的长期机会，但要有耐心
- 在组合中加入抗通胀资产，做好风险管理

**如果你是决策者：**
- 加速能源多元化布局，降低单一来源依赖
- 建立战略能源储备，应对地缘政治风险
- 推动能源效率提升，减少单位GDP能耗

**如果你是普通读者：**
- 理解油价上涨对生活成本的影响，做好预算规划
- 关注能源安全问题，支持可持续能源发展
- 在有限资源下，思考什么是真正的"富足"

### 最后的提醒

> 记住东周的3P原则：科学 ≠ 真理，模型有边界，保持谦卑与好奇。

油价会回落，海峡会开放，但有些东西已经改变。在不确定的世界中，保持清醒的头脑，做好应对各种可能的准备。
```

### 不好的结语示例

```markdown
## 结语

总之，油价上涨是一个复杂的问题，涉及很多因素。我们需要关注市场动态，做出明智的决策。
```

**问题：**
- ❌ 没有回顾三个维度
- ❌ 没有具体行动建议
- ❌ 过于笼统，没有价值
